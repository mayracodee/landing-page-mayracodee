# 💜 Portfólio MayraCodee

Portfólio profissional desenvolvido para apresentar minha trajetória como desenvolvedora Front-End, meus principais projetos e os serviços que desenvolvo para presença digital.

O projeto foi construído com foco em uma interface moderna, responsiva e organizada, representando minha identidade profissional como **MayraCodee**.

## ✨ Sobre o projeto

Este portfólio reúne informações sobre minha atuação como desenvolvedora Front-End e apresenta projetos desenvolvidos ao longo da minha evolução na área.

Além de funcionar como apresentação profissional, o site também possui uma área dedicada aos serviços que ofereço, permitindo que possíveis clientes conheçam as soluções disponíveis e entrem em contato para solicitar um projeto.

## 🖥️ Seções

O site é composto por:

- **Hero** — apresentação principal e acesso rápido aos projetos e serviços.
- **Minha abordagem** — princípios que orientam o desenvolvimento dos meus projetos.
- **Projetos** — apresentação dos principais trabalhos desenvolvidos.
- **Serviços** — Mini Site Profissional, Landing Page e Site Institucional.
- **Investimento** — informações iniciais sobre valores, prazos e manutenção.
- **Contato** — canais para orçamento e contato profissional.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Figma

## 📱 Responsividade

O projeto foi desenvolvido para se adaptar a diferentes tamanhos de tela, incluindo computadores, tablets e dispositivos móveis.

Foram utilizados breakpoints específicos para reorganizar conteúdos, cards, tipografia, espaçamentos e outros elementos da interface de acordo com o espaço disponível.

## 💡 O que desenvolvi neste projeto

Durante o desenvolvimento deste portfólio, trabalhei conceitos como:

- Estruturação semântica com HTML.
- Organização e estilização de interfaces com CSS.
- Flexbox e CSS Grid.
- Responsividade e Media Queries.
- Componentização visual de cards e seções.
- Hierarquia visual e organização de conteúdo.
- Desenvolvimento de uma identidade visual consistente.
